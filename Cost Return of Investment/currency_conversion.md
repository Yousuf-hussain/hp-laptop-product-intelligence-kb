# Currency Conversion Reference

## Purpose

This document provides standard currency conversion rules used across all cost, procurement, support, ROI, business value, and investment analysis reports.

It ensures that all financial calculations generated by the Cost, Business Value, and ROI Analysis Agent are presented consistently in both:

* United States Dollar (USD)
* Indian Rupee (INR)

This enables management, procurement teams, finance teams, customer success teams, and global stakeholders to review financial data using a common reporting standard.

# Conversion Formulas

## USD to INR

Formula:

INR Value = USD Value × Exchange Rate

Example:

USD Cost:

$1,000

Calculation:

1,000 × 83

Result:

₹83,000

---

## INR to USD

Formula:

USD Value = INR Value ÷ Exchange Rate

Example:

INR Cost:

₹166,000

Calculation:

166,000 ÷ 83

Result:

$2,000

# Reporting Rules

## Rule 1

All financial reports must display values in:

* USD
* INR

Example:

| Metric           |    USD |      INR |
| ---------------- | -----: | -------: |
| Hardware Cost    | $3,000 | ₹249,000 |
| Procurement Cost |   $500 |  ₹41,500 |
| Support Cost     |   $300 |  ₹24,900 |

---

## Rule 2

Executive summaries must always show both currencies.

Example:

Total Estimated Cost:

$4,000

₹332,000

---

## Rule 3

ROI calculations should use the same currency throughout the calculation.

Example:

ROI Calculation:

Total Cost:

$4,000

Total Benefit:

$10,000

ROI:

150%

Do not mix USD and INR within the same formula.

---

## Rule 4

Final reports must include a currency disclaimer.

Example:

"All costs shown are estimated using the standard exchange rate defined in the Currency Conversion Knowledge Base."

# Currency Validation Rules

## Valid Conversion

Condition:

Exchange Rate Available

Action:

Convert automatically.

---

## Missing Exchange Rate

Condition:

Exchange Rate not available.

Action:

Display:

"Currency conversion unavailable. Financial values shown in source currency only."

---

## Stale Exchange Rate

Condition:

Exchange rate older than 90 days.

Action:

Display warning:

"Exchange rate requires validation before final financial approval."

# Executive Reporting Rules

For management-facing reports always display:

1. Total Cost (USD and INR)
2. Total Benefit (USD and INR)
3. Net Benefit (USD and INR)
4. Opportunity Cost (USD and INR)
5. Procurement Cost (USD and INR)
6. Support Cost (USD and INR)
7. Total Cost of Ownership (USD and INR)
8. ROI Percentage
9. Payback Period

---

# Agent Usage Rules

The Cost & ROI Analysis Agent must:

1. Use the exchange rate defined in this document.
2. Display all financial values in both USD and INR.
3. Use consistent exchange rates throughout the report.
4. Convert cost, benefit, procurement, support, maintenance, ROI, and opportunity cost values.
5. Display the exchange rate used in the report.
6. Include conversion assumptions in the Executive Summary.
