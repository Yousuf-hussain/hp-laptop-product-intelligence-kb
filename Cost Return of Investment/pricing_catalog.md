# Pricing Catalog

## Purpose

This document provides standard pricing assumptions for HP laptop requirement cost analysis.

It is used by the Cost, Business Value, and ROI Analysis Agent to estimate:

* Base laptop cost
* Hardware upgrade cost
* Configuration cost
* Component cost
* Replacement cost
* Total hardware cost
* Cost comparison between HP laptop families
* Cost impact of customer-specific laptop requirements

All prices are indicative estimates for analysis purposes only and must be validated by the finance, procurement, or product pricing team before final approval.

# Agent Usage Rules

The Cost & ROI Analysis Agent must use this pricing catalog to:

1. Identify the recommended laptop family.
2. Select the closest pricing tier.
3. Add upgrade costs when required.
4. Calculate total hardware cost.
5. Convert USD to INR using the currency rule.
6. Classify cost level.
7. Identify required approval level.
8. Clearly mark all costs as estimates unless verified by procurement or finance.
