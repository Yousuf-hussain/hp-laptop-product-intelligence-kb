# HP Laptop Product Intelligence Knowledge Base

This repository serves as the central source of truth for HP laptop product intelligence.

Contents include:

- HP laptop specifications
- Competitor intelligence
- Hardware feasibility rules
- Firmware feasibility rules
- Inventory guidelines
- Procurement guidelines
- Demand trends

This repository is maintained by Product, Sales, Procurement, Inventory, and Engineering teams and is consumed by AI agents for requirement analysis, product matching, competitor benchmarking, feasibility assessment, and development planning.
